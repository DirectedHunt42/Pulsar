### Setup
1. Download `Pulsar_setup.exe` from the latest release
2. Run the installer to install the program
3. Launch Pulsar from your desktop or start menu
4. Click **Add Income** or **Add Spending** to record your first transaction
5. Use the *Select Date* calendar to backdate entries or plan ahead
6. Watch the graph automatically update with trend lines and gradient shading
7. Use the **Export** button to back up your data or move it to a spreadsheet application
